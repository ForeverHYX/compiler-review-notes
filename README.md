# 编译原理期末复习教程

这套资料按“课程优先版”组织：详细复习课程计划中的 `1-14` 章和 `18 Loop Optimizations`，把教材中未列入教学计划的高级章节作为低优先级拓展速读。正文中文讲解，所有章节都带“术语中英对照”，方便应对英文试卷。

## 推荐使用方式

1. 先读 `00_复习路线与课程覆盖.md`，确认范围和优先级。
2. 按 `01` 到 `18` 顺序学习主线。语法分析、活跃变量、寄存器分配、垃圾回收和循环优化需要手算。
3. 每章先独立完成练习，再对照 `23_练习参考答案.md`。答案重点看判断依据，不要只背最终结果。
4. 考前用 `20_综合练习题.md` 做题，用 `21_速查表.md` 查公式、算法和英文术语。
5. 用 `24_回忆卷解析与考点加固.md` 校准判断题、单选题和大题的考法。
6. 时间充足再读 `19_教材后半拓展速读.md`。

## 浏览器阅读器

本地启动：

```bash
python3 reader_server.py --host 127.0.0.1 --port 8000
```

然后打开：

```text
http://127.0.0.1:8000
```

阅读器会列出所有编号笔记和 `materials/` 里的 PDF。各章“练习参考答案”会在当前页面就地展开；`20_综合练习题.md` 会在每道题标题下提供对应题号的参考答案展开按钮。

当前阅读器是无外部依赖的 Python 标准库实现，方便在本机或服务器上直接运行。笔记内容仍在继续按 PPT、虎书和公开课程资料做初学者向扩充。

如果部署在已有站点的子路径下，例如 `/compiler-notes/`，启动时加：

```bash
python3 reader_server.py --host 127.0.0.1 --port 8010 --base-path /compiler-notes
```

## 例题和答案使用原则

- 每章例题默认只使用当前章以及之前章节已经讲过的知识。
- 如果例题里出现后续章节的表示法，会明确标成“预览版”；预览内容只帮助建立直觉，不要求在当前章掌握格式。
- 练习答案集中放在 `23_练习参考答案.md`，各章练习区都提供跳转入口。编译原理题常有多种合法写法，复习时以语义约束和算法步骤是否正确为准。

## 术语学习原则

考试是英文时，不能只背中文概念。每章末尾的术语表按“英文术语 -> 中文含义 -> 考试提示”组织。复习时建议做到：

- 看到英文能解释中文含义。
- 看到中文能写出常见英文说法。
- 能用英文术语描述算法步骤，例如 `compute FIRST sets`、`construct LR(0) item sets`、`build interference graph`。
