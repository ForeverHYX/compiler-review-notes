# 20 综合练习题

## 使用方法

这些题按考试常见能力组织。建议先不看提示，完整手算一遍，再对照各章讲义检查步骤。

## A. 词法分析

### 题 1：Token 切分

规则：

```text
"if"              IF
"else"            ELSE
[A-Za-z][A-Za-z0-9]* ID
[0-9]+            NUM
"=="              EQ
"="               ASSIGN
[ \t\n]+          skip
.                 ERROR
```

输入：

```text
ifx == if else2 = 123
```

要求：按最长匹配和规则优先级写出 Token 流。

### 题 2：NFA 到 DFA

对正则表达式：

```text
(a|b)*abb
```

要求：

1. 写出 Thompson NFA 的结构。
2. 计算 epsilon-closure。
3. 用子集构造列出 DFA 状态。
4. 标出终态。

### 题 3：DFA 最小化

给一个 DFA 后，按以下模板作答：

```text
初始划分: {final states} | {non-final states}
第 1 轮拆分:
第 2 轮拆分:
最终划分:
最小化 DFA:
```

## B. 语法分析

### 题 4：FIRST/FOLLOW

文法：

```text
S -> A B
A -> a A | epsilon
B -> b B | c
```

要求：

1. 计算 Nullable。
2. 计算 FIRST。
3. 计算 FOLLOW。
4. 构造 LL(1) 表。

### 题 5：消除左递归

文法：

```text
E -> E + T | T
T -> T * F | F
F -> id | ( E )
```

要求：消除直接左递归，并说明改写后是否适合递归下降。

### 题 6：LR(0) 项集

文法：

```text
S' -> S EOF
S  -> E
E  -> E + T | T
T  -> id
```

要求：

1. 构造 `I0 = closure(S' -> . S EOF)`。
2. 计算所有非空 `goto(I, X)`。
3. 画 LR(0) DFA。
4. 写 ACTION/GOTO 表。
5. 用表分析 `id + id EOF`。

### 题 7：SLR 与 LR(1)

给一个发生 LR(0) 冲突的状态：

```text
E -> T .
T -> T . * F
```

要求：

1. 说明 LR(0) 为什么可能冲突。
2. 说明 SLR 如何用 FOLLOW 限制归约。
3. 写出 LR(1) item 如何更精确。

## C. AST 与语义分析

### 题 8：AST 构造

表达式：

```text
a[i] := b + c * 2
```

要求：

1. 画 AST。
2. 写出可能的 AST 节点类型。
3. 指出哪些节点需要 source position。

### 题 9：符号表环境

程序：

```text
var x:int
let
  var y:string
  var x:string
in
  x
end
```

要求：写出进入和退出作用域时的 environment 变化，并说明 `in x` 绑定到哪个声明。

### 题 10：类型检查

判断以下声明是否合法：

```text
type A = {x:int}
type B = {x:int}
var a : A := nil
var b : B := a
```

要求：用 name equivalence 和 nil 规则解释。

## D. IR 与后端

### 题 11：IR 生成

源程序：

```text
while i < n do i := i + 1
```

要求：生成带 `LABEL`、`CJUMP`、`JUMP` 的 Tree IR 结构。

### 题 12：规范化

给表达式：

```text
BINOP(PLUS, ESEQ(MOVE(TEMP a, CONST 1), TEMP a), TEMP b)
```

要求：改写为无 `ESEQ` 的语句序列和表达式。

### 题 13：Basic Block 与 Trace

给线性 IR：

```text
LABEL L1
CJUMP(<, a, b, L2, L3)
LABEL L2
MOVE(x, 1)
JUMP L4
LABEL L3
MOVE(x, 0)
LABEL L4
```

要求：

1. 补齐必要 jump。
2. 划分 basic blocks。
3. 给一种 trace schedule。

### 题 14：Tiling

IR：

```text
MEM(BINOP(PLUS, TEMP fp, CONST -4))
```

目标机器支持 `load r, k(base)`。要求：

1. 画出 IR 树。
2. 说明一条大 tile 如何覆盖。
3. 如果机器不支持 offset addressing，需要几条指令？

## E. 活跃变量与寄存器分配

### 题 15：Liveness

代码：

```text
1: a := 1
2: b := a + 1
3: c := b + a
4: return c
```

要求：

1. 写出每行 use/def。
2. 计算 in/out。
3. 构造 interference graph。

### 题 16：图着色

给干涉图：

```text
a: b c
b: a c d
c: a b d
d: b c
```

要求：用 `K=3` 执行 simplify/select，并给出一种着色。

### 题 17：Coalescing

有 move：

```text
a := b
```

要求：

1. 说明何时可以删除该 move。
2. 说明为什么不能无条件合并 `a` 和 `b`。
3. 简述 Briggs 或 George 判据的保守思想。

## F. 运行时与高级主题

### 题 18：Static Link

三层嵌套函数中，最内层函数访问最外层变量。要求画出 static link 链，并写出访问步骤。

### 题 19：GC

给堆图：

```text
roots -> A -> B
C -> D
D -> C
```

要求：

1. Mark-and-sweep 会回收哪些对象？
2. Reference counting 是否能回收 `C/D`？
3. Copying GC 后 roots 如何更新？

### 题 20：OO Layout

类：

```text
class A { var a; method f }
class B extends A { var b; override f; method g }
```

要求：

1. 画 `B` 对象布局。
2. 画 `A` 和 `B` 的 vtable。
3. 写出动态调用 `x.f()` 的步骤。

### 题 21：Loop Optimization

循环：

```c
for (i = 0; i < n; i++) {
  t = a + b;
  x = 4 * i;
  y = arr[x];
}
```

要求：

1. 找 loop-invariant computation。
2. 找 induction variables。
3. 做 strength reduction。
4. 说明 bounds check 是否可能消除，需要哪些条件。
