# 22 覆盖审计与补强说明

## 结论

当前资料采用课程优先范围：`ch1-ch14` 和教学计划列出的 `ch18` 作为主线，`ch15-ch17`、`ch19-ch21` 作为拓展速读。按课件目录、教材目录和已提取的覆盖清单检查后，主线知识点的覆盖宽度是够的。

但初版在部分手算型算法上偏提纲，不足以保证零基础学生第一次读就能独立做题。本轮已补强高风险章节：

| 文件 | 补强内容 |
|---|---|
| `04_LL1_自顶向下分析.md` | 增加 Nullable/FIRST/FOLLOW 迭代算法和 LL(1) 判断清单 |
| `05_LR0_SLR_自底向上分析.md` | 增加 LR 项集、DFA、ACTION/GOTO、状态栈模拟的手算流程 |
| `08_语义分析_符号表_类型检查.md` | 增加哈希表+scope marker 栈实现、TEnv/VEnv、transExp/transVar/transDec/transTy、two-pass 模板 |
| `10_IR_Tree_中间代码生成.md` | 增加 Ex/Nx/Cx 转换、patch list/backpatching、for 循环翻译 |
| `14_寄存器分配.md` | 增加 worklist 集合、move 分类、图着色手算模板 |
| `18_循环优化_教学计划补齐.md` | 增加 loop/cycle/reducible graph、dominator 迭代、natural loop、loop-nest tree、preheader、LICM safe hoisting 三条件和 strength reduction |

另外，本轮针对复习可用性补了三类问题：

- `01_编译器全景.md` 的例题改为只要求判断各阶段关注点，把正式 Token、AST、IR 等输出放回后续章节学习。
- `23_练习参考答案.md` 集中给出 `01-20` 的练习参考答案，并强调多解题以语义约束和算法步骤为准。
- `01-20` 各章练习区都加入参考答案入口，README 和学习路线也补充“先独立做题，再对答案”的使用规则。
- 重新抽查带例题章节，除第 01 章的超前输出已改为预览外，其余例题均只依赖当前章或前置章节知识。

本次深度扩充进一步把资料从“提纲型复习”改为“可手算学习”的讲义，重点补齐：

- `02` 的 NFA/DFA 五元组、Thompson 构造、子集构造、DFA 最小化、Lex 最长匹配实现。
- `03-07` 的 parse tree、LL(1) 表和栈模拟、LR(0)/SLR 项集与 ACTION/GOTO、LR(1)/LALR/Yacc、Tiger 风格 AST 节点。
- `08-12` 的 Tiger 类型规则、static link 传递、AST 到 IR 翻译规则、canonicalization/basic block/trace、Maximal Munch 和 DP tiling。
- `13-18` 的 liveness 迭代表、图着色和 spill 重写、GC 堆图模拟、OO vtable/offset、dominator/natural loop/LICM/strength reduction/bounds-check。
- `19/21/23` 的拓展例题、速查表和 worked solution 答案规范。
- `24_回忆卷解析与考点加固.md` 根据回忆卷补充判断题、单选题和大题作答模板，用来校准真实考试题型。
- `01-19` 每章都新增统一的“本章考试能力清单”，明确概念题、手算题/构造题、实现题和英文术语题的掌握标准，避免只有个别章节达到详细讲义标准。

## 覆盖范围判断

### A 类：应作为考试主线掌握

- 编译器阶段、Tiger 编译器流水线。
- 词法分析：RE、NFA、DFA、子集构造、DFA 最小化、Lex 冲突规则。
- 语法分析：CFG、推导、二义性、LL(1)、LR(0)、SLR、LR(1)、LALR、Yacc。
- AST 与语义分析：语义动作、符号表、作用域、Tiger 类型规则。
- 运行时和后端：栈帧、static link、IR Tree、规范化、basic blocks、trace、tiling、liveness、register allocation。
- 高级主线：GC、OO、Loop Optimizations。

### B 类：理解即可，不应抢主线时间

- ch13 中 generational/incremental GC 的细节。
- ch14 中多继承字段图着色、membership test、private/classless language、OO 优化。
- ch15-ch17、ch19-ch21 的高级专题。

## 是否“能学明白”

现在每个主线章节至少包含：

- 问题背景：为什么需要这个阶段。
- 核心概念：定义和直觉。
- 算法模板：考试能按步骤写。
- 例题或小例子：把抽象概念落地。
- 常见误区：避免高频丢分。
- 练习题：用于自测。
- 术语中英对照：适配英文考试。

如果时间有限，优先把下面 8 类题练到能手写：

1. NFA 到 DFA 和 DFA 最小化。
2. FIRST/FOLLOW 和 LL(1) 表。
3. LR(0) 项集族、SLR 表、LR 分析栈。
4. 符号表作用域和 Tiger 类型检查。
5. AST 到 IR，尤其 `Ex/Nx/Cx` 和短路求值。
6. Liveness 的 `use/def/in/out` 表。
7. 干涉图和图着色寄存器分配。
8. GC 堆图、vtable 布局、dominator/natural loop/LICM。

## 残余风险

- 课程后续如果新增 PPT，尤其 `ch18` 之外的新优化内容，需要再对照新增课件补一轮。
- 当前资料是讲义和练习提纲，不是完整题库。真正考试准备还需要把 `20_综合练习题.md` 的题目做完，并把不会的题回填到对应章节。
